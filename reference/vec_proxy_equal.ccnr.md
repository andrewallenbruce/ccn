# Comparison

Double dispatch methods to support
[`vctrs::vec_proxy_equal()`](https://vctrs.r-lib.org/reference/vec_proxy_equal.html).

## Usage

``` r
# S3 method for class 'ccnr'
vec_proxy_equal(x, ...)
```

## Arguments

- x:

  A vector x.

- ...:

  These dots are for future extensions and must be empty.
