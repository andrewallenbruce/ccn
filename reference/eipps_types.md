# IPPS-Excluded Hospital Unit Types/Swing-Bed Approvals

IPPS-Excluded Hospital Unit Types/Swing-Bed Approvals

## Usage

``` r
eipps_types
```

## Format

### `eipps_types`

A data frame with 3 rows and 3 columns:

- code:

  IPPS-Excluded type code

- abbr:

  IPPS-Excluded type abbreviation

- desc:

  IPPS-Excluded type description
