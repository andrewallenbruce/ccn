# CCN Medicaid-Only Facility Types

CCN Medicaid-Only Facility Types

## Usage

``` r
medicaid_types
```

## Format

### `medicaid_types`

A data frame with 9 rows and 3 columns:

- code:

  Medicaid-only facility type code

- abbr:

  Medicaid-only facility type abbreviation

- desc:

  Medicaid-only facility type description
