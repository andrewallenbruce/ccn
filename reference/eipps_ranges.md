# IPPS-Excluded Hospital Unit Types/Swing-Bed Approvals

IPPS-Excluded Hospital Unit Types/Swing-Bed Approvals

## Usage

``` r
eipps_ranges
```

## Format

### `eipps_ranges`

A data frame with 3 rows and 3 columns:

- start:

  IPPS-Excluded range start

- end:

  IPPS-Excluded range end

- range:

  IPPS-Excluded range label

- abbr:

  IPPS-Excluded abbreviation

- desc:

  IPPS-Excluded description
