# State Code Lookup Convert state codes to standard state names.

State Code Lookup Convert state codes to standard state names.

## Usage

``` r
get_state_name(x)
```

## Arguments

- x:

  character vector of state codes.

## Value

character vector of state names.

## Examples

``` r
get_state_name(c("00", "01", "83", "A0"))
#> [1] "Arizona"    "Alabama"    "New Jersey" "California"
```
