# CCN Medicare Facility Ranges

- Section **2779A1**

- Title: "CCN for Medicare Providers"

- Pgs. 405-406

- Rev. 226

- Iss./Eff./Impl. 12-06-24

## Usage

``` r
medicare_ranges
```

## Format

### `medicare_ranges`

A data frame with 39 rows and 5 columns:

- start:

  Medicare facility range start

- end:

  Medicare facility range end

- range:

  Medicare facility range label

- abbr:

  Medicare facility abbreviation

- desc:

  Medicare facility description
