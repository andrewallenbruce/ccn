# CCN State Codes

- Section **2779A1**:

- Title: "CCN for Medicare Providers"

- Pgs. 403-404

- Rev. 226

- Iss./Eff./Impl. 12-06-24

## Usage

``` r
state_codes
```

## Format

### `state_codes`

A data frame with 108 rows and 3 columns:

- code:

  State code

- abbr:

  State abbreviation

- name:

  State full name
