# CCN State Codes

From Section 2779A1: CCN for Medicare Providers (pgs. 403-404) (Rev.
226; Issued: 12-06-24; Effective: 12-06-24; Implementation: 12-06-24)

## Usage

``` r
state_codes
```

## Format

### `state_codes`

A data frame with 108 rows and 3 columns:

- code:

  State code

- abbr:

  State abbreviation

- name:

  State full name
