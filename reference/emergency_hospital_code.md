# Emergency Hospital Code Lookup Convert emergency codes to standard emergency names.

Emergency Hospital Code Lookup Convert emergency codes to standard
emergency names.

## Usage

``` r
emergency_hospital_code(x)
```

## Arguments

- x:

  character vector of emergency codes.

## Value

character vector of emergency names.

## Examples

``` r
emergency_hospital_code(c("E", "F"))
#> [1] "Non-Federal Emergency Hospital" "Federal Emergency Hospital"    
```
